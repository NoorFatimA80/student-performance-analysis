# Student Performance Analysis System
A web-based Probability and Statistics semester project developed in Python to analyze and predict student academic performance using statistical methods and regression analysis.
## Project Overview
The Student Performance Analysis System (SPAS) is developed to study the factors affecting student performance such as:

* Study hours
* Attendance
* Sleep hours
* Internet usage
* Previous GPA
The system analyzes student data and predicts final marks using statistical techniques and machine learning.
The project also provides graphs, probability calculations, and performance analysis through an interactive interface.
## Main Features

* Student marks prediction
* Pass/fail probability analysis
* Statistical calculations
* Graphical representation of data
* Linear regression model
* Interactive web application
